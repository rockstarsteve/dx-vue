#### dx-vue-first

 - vue 基本
    - data：数据定义
    - computed ：计算属性，对一些值进行监控
    - methods：方法，
    - watch：侦听器，对指定值进行值的监控
 - vuex 
    - state：基本数据
    - mutations：提交更改数据的方法，同步
    - actions：像一个装饰器，包裹mutations，使之可以异步
    - getters ：从基本数据派生的数据 
    - modules ：模块化Vuex
